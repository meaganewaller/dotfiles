# Contributing

Thank you for investing to your time in contributing to this project! Any contribution you make will be reflected in the README.

When contributing to this repository, please first discuss the change you wish to make via issue, email, or any other method with the owners of this repository before making a change.

Read and follow the [Code of Conduct](#) to keep this community approachable, kind, and inclusive.

## Pull Request Process

1. Fork this repository
2. Create a new branch in your fork repository, for example **patch-1**
3. Make changes with clear commit descriptions
4. Open a new pull request in this repository
5. Wait for the owner/maintainers to review
6. Upon approval, your PR will be merged into the code base
7. Thanks for your contribution!
