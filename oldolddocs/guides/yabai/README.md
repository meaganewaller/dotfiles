### 🧙‍♂️<samp> Yabai Wizardry </samp>🔮

Yabai is your wand, and here's how to cast spells:

**1. Clone the repo**

 ```sh
 💲 git clone git@github.com:meaganewaller/dotfiles.git ~/.dotfiles
 ```

**2. Enter the Wizard's Lair:**

 ```sh
💲 cd ~/.dotfiles
 ```

**3. Run the Yabai Wizard:**

 ```sh
💲 rake yabai:install
 ```

 Yabai shall now heed your commands, reshaping your windows at will. Let the magic begin! ✨

**4. Unlock Hidden Powers:**

 ```sh
💲 rake yabai:unleash
 ```

 Discover advanced spells and customizations to master your desktop domain.

 🌟 Remember, with great window management comes great responsibility—or maybe just more screen real estate.
