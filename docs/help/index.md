# Getting Help

There are a few ways to get help with the dotfiles. However, because these are
my personal configurations, and they're given with no warranty, I can't
provide any technical support. I'm happy to answer questions, but I can't guarantee that I'll be able to help you. In lieu of technical support, I've provided a few ways to get help.

## Resources

Exisiting resources are the best way to get help. I've tried to document
everything as thoroughly as possible, and I've provided a few resources to
help you get started.

- [Frequently Asked Questions](/help/faq)
- [Troubleshooting](/guides/troubleshooting)
- [Glossary](/docs/glossary)
- [Guides](/docs/guides)

## Issues

If you're having a problem with the dotfiles, you can [open an issue](
https://github.com/meaganewaller/dotfiles/issues/new) on GitHub. Please be
sure to include as much information as possible, I've created a template
[templates](https://github.com/meaganewaller/dotfiles/blob/main/.github/ISSUE_TEMPLATE.md)
for you to use.
